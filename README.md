## 二人斗地主算法规则介绍
#### 发牌  
牌库中去除3，4，每人发17张牌，留3张底牌，其余9张作为废牌。确定地主后底牌归地主所有。
#### 叫地主与抢地主  
随机指定两位玩家谁先叫地主，另一名玩家可以抢地主，最多可来回抢4次地主。
#### 让牌  
叫地主默认让1张牌给农民，每抢一次地主让牌数+1，最多抢4次地主，所以最多让5张牌。
农民出牌后，剩余手牌数小于或等于让牌数，即获得胜利。

## 算法介绍
* 基于 [Douzero](https://github.com/kwai/DouZero) 修改  
* dmc算法对于二人斗地主更加适用  
* 效果：高级场胜率49%，专家场胜率47%
